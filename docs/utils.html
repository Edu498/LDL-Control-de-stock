def execute_query(query, params=None, fetch_one=False, fetch_all=False, commit=False): 
124def execute_query(query, params=None, fetch_one=False, fetch_all=False, commit=False):
125    """
126    Ejecuta una consulta SQL genérica, maneja la transacción y cierra recursos opcionalmente.
127
128    Args:
129        query (str): Sentencia SQL a ejecutar.
130        params (tuple, opcional): Parámetros a inyectar en la consulta.
131        fetch_one (bool, opcional): Si es True, retorna solo la primera fila de resultados.
132        fetch_all (bool, opcional): Si es True, retorna todas las filas de resultados.
133        commit (bool, opcional): Si es True, ejecuta conn.commit() tras ejecutar la sentencia SQL.
134
135    Returns:
136        dict, list o mysql.connector.cursor: Depende de los parámetros:
137            - dict si fetch_one es True (gracias a dictionary=True).
138            - list de dicts si fetch_all es True.
139            - mysql.connector.cursor si ambos son False (debe ser cerrado manualmente junto con la conexión).
140
141    Raises:
142        mysql.connector.Error: Si ocurre un error en la base de datos (se ejecuta Rollback si conn existe).
143    """
144    conn = None
145    cursor = None
146    try:
147        conn = get_connection()
148        cursor = conn.cursor(dictionary=True)
149        cursor.execute(query, params or ())
150        
151        if commit:
152            conn.commit()
153        
154        if fetch_one:
155            return cursor.fetchone()
156        elif fetch_all:
157            return cursor.fetchall()
158        else:
159            return cursor
160        
161    except Error as e:
162        if conn:
163            conn.rollback()
164        raise e
165    finally:
166        if cursor and not fetch_one and not fetch_all:
167            cursor.close()
168        if conn and not fetch_one and not fetch_all:
169            conn.close()

Ejecuta una consulta SQL genérica, maneja la transacción y cierra recursos opcionalmente.

Args: query (str): Sentencia SQL a ejecutar. params (tuple, opcional): Parámetros a inyectar en la consulta. fetch_one (bool, opcional): Si es True, retorna solo la primera fila de resultados. fetch_all (bool, opcional): Si es True, retorna todas las filas de resultados. commit (bool, opcional): Si es True, ejecuta conn.commit() tras ejecutar la sentencia SQL.

Returns: dict, list o mysql.connector.cursor: Depende de los parámetros: - dict si fetch_one es True (gracias a dictionary=True). - list de dicts si fetch_all es True. - mysql.connector.cursor si ambos son False (debe ser cerrado manualmente junto con la conexión).

Raises: mysql.connector.Error: Si ocurre un error en la base de datos (se ejecuta Rollback si conn existe).

 53                bg='#FFF3CD', fg='#DC3545').pack()

 55        tk.Label(frame_titulo, text="¡ALERTA DE STOCK CRÍTICO!", 
 56                font=("Segoe UI", 16, "bold"), bg='#FFF3CD', fg='#856404').pack()
 57        
 58        # Resumen
 59        frame_resumen = tk.Frame(ventana, bg='#FFF3CD')
 60        frame_resumen.pack(pady=10)

 62        tk.Label(frame_resumen, text=f"🔴 Sin stock: {len(sin_stock)} productos", 
 63                font=("Segoe UI", 11), bg='#FFF3CD', fg='#DC3545').pack()
 64        tk.Label(frame_resumen, text=f"🟡 Stock bajo: {len(stock_bajo)} productos", 
 65                font=("Segoe UI", 11), bg='#FFF3CD', fg='#FFC107').pack()
 66        
 67        # Frame para lista

 83                    font=("Segoe UI", 12, "bold"), bg='#FFF3CD', fg='#DC3545').pack(anchor=tk.W, pady=(10,5))
 84            
 85            for p in sin_stock:
 86                frame_prod = tk.Frame(scrollable_frame, bg='#FFF3CD')
 87                frame_prod.pack(fill=tk.X, padx=10, pady=2)

 89                tk.Label(frame_prod, text=f"• {p['nombre']}",
